# General Manager App

A Spring Boot 3.x application to manage **employees** and **persons** with secure authentication and authorization using **Keycloak** and **JWT OAuth2**.

---

## Features

- CRUD operations for **Employees** and **Persons**
- Authentication & Authorization via **Keycloak**
- Built with **Java 21** and Spring Boot 3.x
- PostgreSQL database (can be replaced by H2 for testing)
- RESTful API endpoints

---

## Tech Stack

- **Backend:** Java 21, Spring Boot 3.x, Spring Security, Spring Data JPA  
- **Authentication:** Keycloak, OAuth2, JWT  
- **Database:** PostgreSQL  
- **Build Tool:** Maven  
- **Testing:** JUnit 5, Mockito  

---
